### About

I build AI-powered applications, mostly around computer vision and vision-language models. Recent work includes a real-time camera that answers questions about what it sees, weather emulation experiments, and a land-use optimizer. Currently going deeper into deep learning and LLMs.

Open to collaborating on anything vision or ML related.
